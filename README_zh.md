# 带边框的涂蜡与虫蚀方块

用负法线模型给涂蜡方块和虫蚀方块加上边框！兼容资源包！

---

## 涂蜡与虫蚀方块拥有更加容易辨认的物品形式

灵感来源于 [XeKr 的方纹](https://www.mczwlt.net/resource/lsd5dbu4)，使用负模型制作边框。

本包仅修改物品形式：GUI中的物品、掉落物、展示框中的物品。 **其方块形式保持不变。**

为了使风格统一，木门、铁门、未涂蜡的铜门也拥有和涂蜡的铜门相仿的3D物品模型。
